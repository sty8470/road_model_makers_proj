<데이터 포맷 (벡터 데이터)>

shp
  - 기하학적 위치 정보(도형 정보)를 포함하고 있는 벡터 파일
sbn, sbx, shx
  - 공간적(sbn, sbx), 기하학적(shx) 인덱스 파일
  - shp 파일에 담긴 도형 정보의 위치를 얻을 수 있음
dbf
  - 속성정보를 포함하고 있는 데이터베이스 파일
  - shp에 있는 도형 정보에 대한 속성 정보를 담고 있음
  - shp에 있는 도형 정보 순서에 맞게 dbf에 속성 정보가 저장되어 있음
  - dBase IV구조로 되어 있음
prj
  - 투영, 좌표계 정보를 포함하는 프로젝션 파일
